About IFAS.io

IFAS (Information Feed Analysis System) is a project sponsored by HKCERT and developed by HKCERT and CSIRT Foundry

IFAS is designed to gather security event logs from many public sources, normalise them to similarly-structured event data, store them, and allow searching and analysis of events.

IFAS can answer these questions:

How many phishing attacks against Australia were recorded in the last 3 months?

Which ISP in Peru has the highest number of drones?

Show me TLDs grouped by number of C&Cs hosted, then by country code, then by ASN.

Tell me when any events are detected with a tld of *.go.jp.

and more...

The Architecture of IFAS

Show Diagram


Abusehelper gather, process and enrich feeds; generate events
Logstash process and normalize feeds
Elasticsearch store events in schema-free index server
Kibana search through events
IFAS Reporter get overall statistics; build realtime dashboards

All the data is stored LOCALLY!
